# Task Plan: 条件検索機能の追加

## 現状の課題と目的
- 現行の機能（テキスト抽出・ファイルアップロードからの検索）を維持したまま、キーワードや条件（例：滋賀県、橋長10m以下、市町村管理）のみで施設を検索できる「条件検索機能」を追加する。
- プレビュー上でリストを表示し、CSVおよびQGIS（GeoJSON）にエクスポート可能にする。

## 実装ステップ

### 1. UIの設計（index.html）

- [x] 「条件から検索」タブ内に以下のフォームを配置：
  - [x] 対象施設の種類（セレクトボックス・必須）

  - [x] レスポンスに含まれる「すべての基礎データ」を対象に横断検索できるキーワード入力欄（スペース区切りで複数指定可能にする）
  - [x] 個別の条件（市区町村、管理者、路線名、施設名、寸法範囲、判定区分（点検結果）など）
